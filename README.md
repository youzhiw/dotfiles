# Neovim notes
- remember to install ripgrep for telescope live grep to work

## Neovim kebinds
- `<Leader>ff`: find file
- `<Leader>fg`: live grep
- `<Leader>fb`: find buffer
- `<Leader>ee`: open file tree
- `<Leader>ef`: open file tree under current file directory
